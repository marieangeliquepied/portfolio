À propos

Après un premier parcours professionnel, j'ai choisi de me reconvertir dans le développement web, par goût de comprendre comment les choses fonctionnent et de construire des outils utiles de bout en bout.

Je travaille aujourd'hui sur des applications full stack modernes — React et Next.js côté interface, Node, Laravel et Supabase côté données. J'aime le code soigné, les interfaces claires et les projets qui ont du sens.
